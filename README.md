# 📊 Telecom Subscriber & Churn Analytics (SQL Project)

## 📌 Project Overview
This project is built using Oracle SQL and focuses on analyzing telecom subscriber data. It includes various SQL queries to explore customer behavior, billing information, usage records, and churn risk analysis.

The project demonstrates the use of SQL for data analysis, reporting, and business decision-making.

---

## 📂 Database Tables

- **Subscribers** – Customer information
- **Plans** – Subscription plan details
- **Billings** – Billing and payment records
- **Records** – Call and internet usage records

---

## 🛠 Technologies Used

- Oracle SQL
- Oracle SQL Developer

---

## 📈 SQL Concepts Covered

- SELECT
- WHERE
- ORDER BY
- GROUP BY

- JOINS (INNER, LEFT)
- Aggregate Functions
- CASE WHEN
- DECODE
- NVL
- Subqueries

- Window Functions (RANK, DENSE_RANK, ROW_NUMBER)

## 📊 Analysis Performed

- Subscriber usage analysis
- Payment status localization
- Revenue analysis by plan
- Most popular subscription plans
- Call and internet usage reports
- Churn risk detection
- Average and maximum usage calculations
- Customer categorization
- Ranking reports
